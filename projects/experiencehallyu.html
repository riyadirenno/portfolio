Experience Hallyu.

Creating a Multimedia Responsive Website showcasing a genre of our choice; First introduction to the Bootstrap Framework.

This project showcases the creation of a website showcasing a genre of our choosing. I choose K-Pop as I personally have an interest to the genre and the I believe the genre is associated with many visually appealing features that I’d like to implement for my final piece.

This website is my first website done with the Bootstrap framework. We were given a month worth of lectures of the basics of the framework and we have the rest of the date till submission to complete the website. I use Bootstrap mostly for its grid system as it helps making a responsive website much easier.

The colors scheme I choose for this website represents the colors from the Korean flag. I added a hint of green in some parts of the website as I believe that it gives a slight contrast to the overall look of the website. I added some javascript on the website which allows some animations which I believe compliments the whole look of the website.

The whole project took over 3 months to complete. The first month we were given a crash course of the Bootstrap framework and how it works. By the second month we were given time to ourselves to complete the project till the submission date; I take this time to do my wireframes and code some of the pages I’ve designed. We has a group feedback session at the last month and by then I’ve finalised all the code and make sure everything is working and links to the right place.
